# Alrededor de los EE.UU.

Este es un proyecto que estoy haciendo en el Sprint 6 del bootcamp de TripleTen.

## ¿Qué hace la página?

Es una página donde se muestran fotos de diferentes lugares (como Yosemite y el Lago Louise). En cada foto puedes:

- Darle "me gusta" (❤️)
- Eliminarla (🗑️)

También hay un botón para editar tu perfil (tu nombre y una descripción corta).

## ¿Qué aprendí en este proyecto?

- Cómo separar los datos (nombres y fotos) del HTML, usando un array de JavaScript.
- Cómo crear elementos HTML desde JavaScript, en vez de escribirlos todos a mano.
- Cómo abrir y cerrar una ventana emergente (modal) con clics.
- Cómo hacer que un formulario guarde lo que escribes y actualice la página.

## Tecnologías que usé

- HTML
- CSS
- JavaScript (sin frameworks, solo JavaScript "puro")

## Cómo verlo funcionando

1. Descarga o clona este proyecto.
2. Abre el archivo `index.html` en tu navegador.
3. ¡Listo! Ya puedes interactuar con las tarjetas y el perfil.

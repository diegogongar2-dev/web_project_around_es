# Alrededor de los EE.UU.

Este es un proyecto que estoy haciendo en el Sprint 6 del bootcamp de TripleTen.

## ¿Qué hace la página?

Es una página donde se muestran fotos de diferentes lugares (como Yosemite y el Lago Louise). En cada foto puedes:

- Darle "me gusta" (❤️)
- Eliminarla (🗑️)
- Ampliarla en una ventana emergente al hacer clic sobre la imagen

También puedes:

- Editar tu perfil (nombre y descripción) desde el botón de lápiz.
- Agregar nuevas tarjetas (nombre + enlace a una imagen) desde el botón "+"; la tarjeta nueva aparece como primer elemento de la lista.

## ¿Qué aprendí en este proyecto?

- Cómo separar los datos (nombres y fotos) del HTML, usando un array de JavaScript.
- Cómo generar elementos HTML dinámicamente a partir de un `<template>`, en vez de escribirlos a mano o con `innerHTML`.
- Cómo usar parámetros predeterminados para manejar datos incompletos (título "Sin título" e imagen de repuesto).
- Cómo abrir y cerrar varias ventanas emergentes (modales) reutilizando las mismas funciones `openModal()` / `closeModal()`.
- Cómo capturar el envío de un formulario para crear contenido nuevo en la página.

## Tecnologías que usé

- HTML
- CSS
- JavaScript (sin frameworks, solo JavaScript "puro")

## Cómo verlo funcionando

1. Descarga o clona este proyecto.
2. Abre el archivo `index.html` en tu navegador.
3. ¡Listo! Ya puedes interactuar con las tarjetas y el perfil.
